📋 Requirements

Before running the project, make sure the following are installed on your system:

1. Python
Python 3.8 or higher

Check if Python is installed:

python --version
2. pip (Python Package Manager)
pip --version

pip usually comes installed with Python.

3. Required Python Libraries

Install the required dependencies using:

pip install -r requirements.txt

Typical libraries used in this project include:

streamlit
langchain
google-generativeai
pypdf
faiss-cpu
python-dotenv


⚙️ Installation
1. Clone the repository
git clone https://github.com/arushkumarme/AI-Research-Paper-Analyzer.git
2. Navigate to the project directory
cd AI-Research-Paper-Analyzer
3. Install dependencies
pip install -r requirements.txt


▶️ Running the Application

If the project uses Streamlit, run:

streamlit run app.py

After running the command, the application will open in your browser at:

http://localhost:8501
